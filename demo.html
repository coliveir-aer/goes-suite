GOES Data Viewer — Event Gallery

Notable severe-weather and space-weather events, each opening directly in the viewer with satellite, product, sector, and time pre-selected. Data streams live from the NOAA Open Data archive.

All times UTCSize chips are measured, not estimated ★ = meso sector verified on the event
How this page works

Loops vs frames. Loop links open an animation over the event window; frame links open a single scene. Loops stream while loading — playback starts before the download completes. 5-min cadence links cover a longer window at reduced frame rate for the same data budget class.

★ Verified meso sectors. GOES mesoscale sectors float. For each event the actual meso files from that hour were read and each sector's center computed from its fixed-grid geometry; a star means that sector was confirmed on (or tracking) the event. Where no meso was tasked, the card says so — use CONUS and zoom.

Composite links (double border) open a multi-layer stack — e.g. fire detections over True Color — using the viewer's stack deep-link format.

Sizes. Every button's chip is the summed byte size of the exact file set that window resolves to in the NOAA archive.
